# 🌦️ Weather Check Application

A simple and responsive **Weather Check Application** built using **HTML, CSS, and JavaScript**. This app allows users to check real-time weather information for any city in the world.

---

## 🚀 Features

* 🌍 Search weather by city name
* 🌡️ Displays temperature, humidity, and weather conditions
* 📡 Real-time data using Weather API
* 🎨 Clean and responsive UI
* ⚡ Lightweight and fast (no frameworks used)

---

## 🛠️ Tech Stack

* **HTML** – Structure of the application
* **CSS** – Styling and responsiveness
* **JavaScript** – Logic and API integration

---

## 📂 Project Structure

```
weather-app/
│── index.html
│── style.css
│── script.js
│── assets/ (images/icons)
```

---

## 🔧 How It Works

1. User enters a city name in the input field
2. JavaScript fetches weather data from an external API
3. The app dynamically updates the UI with:

   * Temperature
   * Weather condition
   * Humidity
   * Wind speed (if added)
